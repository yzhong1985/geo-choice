Starbucks Location Smart

This website provides an algorithm to analyze and predict successful locations for Starbucks in Seattle area

Task 1
Visualize current location

  • Click on a current location
  • See location attributes such as distance to road, number of tweets, size of location and number of populated Asian in location tract
  • Customize different demographic visualization and filters for current Starbucks location
  1. See which locations are less than 2500 sq ft and which is more than 2500 sq ft
  2. See locations with different ethnicity
  3. See locations based on different educational levels

Predict future locations

  • Drop the pin for your preference future location
  1. Right click where you want to drop the pin
  2. Add potential location
  3. Right click added potential to visualize different options such as visualize demographic, move potential to a different location, delete potential, and calculate sales for potential
